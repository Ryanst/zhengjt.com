Java集合框架总结

发表于 2014-09-29   |   分类于 Java   |  

JAVA集合的接口和实现分离

Collection接口的类实现了单元素存储、Map接口实现了键/值的映射表(键唯一)
Collection添加用add,设置值用set(index),获取值需要有序用get(index)
Map添加用put(key,value),设置值依然用put(key,value)覆盖原值,获取值用get(key)

队列：LinkedList实现了Queue接口，因此要使用队列的时候使用LinkedList即可

``` java
Queue queue = new LinkedList();
```
Queue接口：

\ | 抛出异常 | 返回特殊值
-—-|—–|—–
插入 | add(e) | offer(e)
移除 | remove() | poll()
检查 | element() | peek()

add:添加到队尾

remove:移除队头元素//与一般的认知有些不同

element:返回队头元素

两列的功能相同，区别在于当队列为空的时候第一列抛出异常，第二列一般只返回null

总结：

  • List：index类型集合，可以用之实现栈、队列等数据结构
  • LinkedList：链表结构

  • ArrayList：有序数组结构

  • Set：无index，储存值类型集合，不能有相同的值

  • HashSet：无序值集合，散列算法，当散列值不同时才加入新值

  • TreeSet：有序值集合，红黑树

  • Map：key/value类型集合，不能有相同的Key

  • HashMap：无序key/value集合
  • TreeMap：有序key/value集合

数组转换成集合比较方便,使用Array.asList的包装器即可

``` java
String []values;
HashSet hashSet = new HashSet\<>(Array.asList(values));
```

集合转换成数组——手动转

Vector、Stack、HashTable都是JAVA集合的基础版本，目前已经被上述集合框架代替

队列还有一个实现是ArrayDeque，可以实现双向队列
